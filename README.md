# DevRoast

Ferramenta para analisar e "assar" código JavaScript/TypeScript. Cole seu código e receba feedback brutalmente honesto sobre a qualidade do código.

## Funcionalidades

- **Análise de Código** - Cole qualquer código JavaScript e receba uma avaliação
- **Modo Roast** - Ative para receber feedback com máximo de sarcasmo
- **Leaderboard** - Veja os códigos mais horrorosos ranqueados por vergonha
- **Visualização de Diferenças** - Veja as sugestões de melhoria com highlight de diff

## Stack

- Next.js 15
- Tailwind CSS
- Shiki (syntax highlighting)
- Base UI

## Como Executar

```bash
npm run dev
```

Acesse http://localhost:3000
